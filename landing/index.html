Data Platform

Data Platform — не про сухі дані. Це візуалізований конструктор даних для доступу до статистичної інформації, необхідної бізнесу для прийняття рішень. Також, платформа є сервісом зручного доступу до даних за допомогою Rest API для розробників програмного забезпечення. Data Platform is not about plain data. It is a visualized data constructor for accessing the statistical information, needed by business to make decisions. The platform is also a service for an easy access to the data, using the Rest API for software developers.

Зрозумілий інтерфейс Clear interface

Зручний інтерфейс навіть для тих, хто ніколи не користувався такими системами. Convenient interface even for those who have never used such systems.

Гнучкі налаштування Flexible settings
